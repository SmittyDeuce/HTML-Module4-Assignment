# Module 4 HTML Assignment

## Overview
This project is a basic HTML webpage demonstrating the use of forms, tables, navigation bars, and images. It follows the assignment requirements to create a structured and semantic HTML5 document.

---

## Features

### 1. Form Creation
- The page contains a **contact form** where users can enter:
  - Name (text input)
  - Email (email input)
  - Message (textarea)
  - A radio button group asking how the user found the site (Google, Social Media, YouTube, Other)
  - A **submit button** to send the form data.

---

### 2. Table Structure
- A **table** is included that displays sample product data.
- The table consists of three columns:
  - **Product Name**
  - **Price**
  - **Quantity**
- Three example products are listed: **Eggs, Ice Cream, and Hamburger.**
- A header row labels each column.

---

### 3. Navigation Bar
- A simple **navigation bar** is implemented using an unordered list (`<ul>`).
- The navigation contains links to:
  - Home
  - About
  - Contact

---

### 4. Adding Images
- Two **placeholder images** are included using `https://placehold.co/300x200`.
- The second image has a **rounded border** applied using the `style
